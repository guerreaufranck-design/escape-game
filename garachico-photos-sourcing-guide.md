## Processus de compilation

### Pour chaque photo trouvée, note :

1. **Titre** : Nom exact du fichier
2. **Auteur** : Photographe/créateur
3. **Licence** : CC-BY, CC-BY-SA, CC0 (noter la version exacte)
4. **URL source** : Page Wikimedia/Flickr
5. **URL téléchargement direct** : Lien vers fichier haute résolution
6. **Dimensions** : Largeur × Hauteur en pixels
7. **Taille fichier** : En MB (optionnel)

### Exemple d'entrée :

```
**Titre :** Castillo_San_Miguel_Garachico_2023.jpg
**Auteur :** John Smith
**Licence :** CC-BY-SA 4.0
**URL source :** https://commons.wikimedia.org/wiki/File:Example.jpg
**Téléchargement :** https://upload.wikimedia.org/wikipedia/commons/thumb/...
**Dimensions :** 4000 × 3000 px
```

## Licences acceptables

✅ **CC0** — Domaine public, utilisation totale liberté
✅ **CC-BY 4.0** — Requiert attribution
✅ **CC-BY-SA 4.0** — Requiert attribution + même licence en partage
❌ **Toute licence restrictive** (ND, NC, etc.)
